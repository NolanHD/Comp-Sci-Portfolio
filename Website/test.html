Introduction to Computer Science

Welcome to Computer Science! Computer Science is a class where people of all levels of experience come to learn or expand their knowledge of computers. From coding to building computers, we do it all!

Main Purpose

In this class we are learning how to use the coding language called HTML and CSS. Along with that, there will be an introduction to the components of a computer and how one is built. We will also delve into how the Linux OS works and how our knowledge of HTML and CSS can apply to it. Hopefully by the end of this interval, everyone will have a completed website using the basic knowledged we have been taught.
