What is HTML?

HTML is a standardized system for tagging text files to achieve font, color, graphic, and hyperlink effects on World Wide Web pages.





        What does it stand for?

         Hypertext Markup Language

         How is it used?

         It's standardized system for tagging text files to achieve font, color, graphic, and hyperlink effects on
         World Wide Web pages.

        Where did it come from?

HTML (HyperText Markup Language) was created by John "Tim" Berners-Lee. Originally created to share work with his colleagues, as the internet evolved a universal language was needed to code the internet and HTML rose to the occasion.




        How is it written?

         It is written by using tags that the browser understands and renders into a webpage.

Tags

What are tags?

Tags are hidden keywords within code that define how a browser renders a page. The tags tell the browser how to format and display the code. Most tags have an opening and closing such as the h1 code but some only have an opening such as the br code.




        How do you write one?

         To write a tag you start with an opening tag written in pointy brackets. After writing your text you close it by
         writing the same tag but with a forward slash before the tag.

What are attributes?

Attributes are codes written inside tags that modify the text inside the opening and closing tag.
