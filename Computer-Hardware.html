Hardware


What is a computer?

To sum it up, a computer is a machine that manipulates data. It can store, retrieve and process data? Its made up of many different parts that each have their own job. A few core parts of a computer are the CPU, RAM, hard drive, keyboard and mouse.

        What is a CPU?

A CPU or a central processing unit is a core part of most computers. The CPU is kind of like the brain of a computer. It sends signals which control the other parts of the computer. The CPU had a list of things it does called "instructions". The picture on the left was from the computer deconstruction in class. The CPU was built into the motherboard, more on that later, and was made by Intel in China





What is RAM?

RAM or random access memory is a type of computer memory that can be accessed randomly. It's used by the system to store data for processing by a computer's central processing unit (CPU), any byte of memory can be accessed without touching the preceding bytes. Out off all the computers, RAM is the most common type of memory and other devices, such as smart phones.



What is a motherboard?

A motherboard is a printed circuit board with places for the primary components of a computer. It also can have connectors for other circuit boards to connect to. Basically it's the centerpiece that holds/connects everything together.


What is a hard drive?

A hard drive is self-contained storage device that has a read-write function. It is used for long term memory and is made of a disk or multiples disks. There are also hard drives called a SSD or a solid state drive. It has no moving parts and the information is stored in microchips not spinning disks.

        How does it work?

Typically has one disk and each side has a magnetic coating. The RPM of the disk or rotations per minute, determine how quickly your computer can access the information stored on these disks. Hard drives with disks can reach 10,000 rpm, but SSD's are still the fastest.




What are computer peripherals?

The most common computer peripherals are the mouse, keyboard and monitor. These control the core functions in the UI and are necessary to use a computer. There are also other peripherals such as speakers or printers and they aren't necessary but add functions to a computer.

How do all the parts of a computer communicate and work with each other?

The CPU, RAM, motherboard, and hard drive are the core of a computer. They are all connected to the motherboard and each have their own functions. The CPU is the brain that tells the other parts what to do. The RAM is used by the system to store data for processing by a computer's central processing unit (CPU). Finally the hard drive reads and writes data for long term storage. Hopefully if all the computer parts come together correctly and work you'll have a computer!
